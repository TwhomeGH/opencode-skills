# context-cleanup

預設啟用的 token 使用效率技能。控制 output tokens、避免重複敘述、減少不必要的工具呼叫。

## 功能

| 類別 | 原則 |
|------|------|
| Output 回覆 | 直接回答，不加前言後語、感謝、總結 |
| 檔案操作 | 不附帶說明文字，diff 只貼必要部分 |
| 錯誤處理 | 摘要行號與類型，完整 stack trace 只在使用者要求時才貼 |
| 工具呼叫 | 優先使用 grep/glob，批次平行呼叫，避免重複搜尋 |
| 任務管理 | 已完成項目從 todo 清單移除，不保留詳細描述 |

## 使用方式

**無需手動啟用** - 每輪回覆自動遵循上述原則。

關鍵字可強化觸發：
- 「省 token」
- 「精簡回覆」
- 「減少輸出」
- 「不要廢話」
