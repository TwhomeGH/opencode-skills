# filter-html-noise

預設啟用的 HTML 雜訊過濾技能。分析 HTML 時自動忽略無語義的視覺雜訊，保留業務邏輯相關內容。

## 功能

| 類別 | 處理方式 |
|------|----------|
| SVG 路徑座標 (`d="M...Z"`) | **忽略** |
| 內聯 SVG 幾何細節 | **忽略**（保留 `<svg>` 標籤與 class/id） |
| Base64 圖片資料 | **忽略** |
| 內聯樣式視覺屬性 | **忽略** |
| 動態 ID 內部巢狀結構 | **忽略**（保留元素本身） |
| 廣告模組結構 (`ytp-ad-`, `ad-`) | **保留分析** |
| 追蹤/分析代碼 | **保留分析** |
| 播放器 UI 功能結構 | **保留分析** |
| 語義文字、連結、表單 | **保留分析** |

## 例外情況

用戶**明確要求**時才分析 SVG 細節：
- 「分析 SVG」
- 「看 SVG 路徑」
- 「檢查圖標幾何」

## 使用方式

**無需手動啟用** - 遇到 HTML 分析任務自動生效。

關鍵字可強化觸發：
- 「過濾 HTML」
- 「忽略 SVG」
- 「去除雜訊」
- 「專注語義」
